# Officeverse Building Sprites

Spriteset generado para estructura de oficina: suelos, paredes, puertas,
ventanas, rodapies y detalles de pared.

- `source-office-building.png`: fuente generada con fondo chroma key.
- `office-building.png`: spritesheet final con transparencia.
- `office-building-preview.png`: preview sobre patron checker.
- `office-building.manifest.json`: frames y `gridSize` para Phaser/editor.

Los objetos usan un grid base de 16x16 y se renderizan con el mismo sistema de
ajuste que los muebles de Officeverse.
